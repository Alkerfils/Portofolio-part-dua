Learning Notes

Command Membuat Directory/Folder.

$ mkdir Nama_Directory
Output tidak akan muncul, Tapi Directory/Folder yang baru saja di buat akan langsung muncul dengan Nama_Directory.

Yang dipelajari: Dasar Command Linux untuk Membuat Directory/Folder.

Command Membuat Directory/Folder yang DiSembunyikan.

$ mkdir .Nama_Directory
Output tidak akan muncul, Tapi Directory/Folder yang baru saja di tidak akan langsung muncul, harus di panggil agar muncul dengan ls -a atau ctrl + h di file manager.

Yang dipelajari: Dasar Command Linux untuk Membuat Directory/Folder yang Bersembunyi.

Command Membuat Directory/Folder dengan Spasi

$ mkdir 'Nama Folder'.
atau
$ mkdir "Nama Folder".
atau
$ mkdir Nama\ Folder.
Output tidak akan muncul, Tapi Directory/Folder yang baru saja dibuat, akan langsung muncul di file manager, nama folder dengan spasi.

Yang dipelajari: Dasar Command Linux untuk Membuat Directory/Folder dengan Spasi.

Command List untuk melihat semua yang ada di Directory.

$ ls Nama_Directory
Output akan Menampilkan List-list yang ada di didalam Directory seperti Folder, File, dll... Contohnya:
Animelist(Folder) | Book.pdf(File) | Catatan.txt Presentasi.pptx(File) | Kasir.cbl(File) | Index.html

Yang dipelajari: Dasar Command Linux untuk Melihat semua isi dari target Directory/Folder.
